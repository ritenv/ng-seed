ng-seed

A base for front-end development with AngularJS

Instructions:
  1. Install bower using sudo npm install bower -g. Then run bower in the ngbp directory to install front-end dependencies: bower install.
  2. Install gulp using sudo npm install gulp -g. Then install the gulp plugins: npm install.
  3. Install Java from here, if prompted: http://support.apple.com/kb/DL1572 (to run Selenium)
  4. Run gulp to compile, minify and lint front-end assets.
  5. Run gulp debug default while you are developing so that you can use the unminified version of your assets.
  6. Run gulp server or gulp debug server if you want the default task run and your files served by a development HTTP server (includes livereload)


Hope this boostrap seed project works well.
