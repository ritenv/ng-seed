ng-seed

A base for front-end development with AngularJS

Instructions:
  1. Install bower using sudo npm install bower -g.
  2. Install gulp using sudo npm install gulp -g.
  3. Install Java from here: http://support.apple.com/kb/DL1572 (to run Selenium)
    Skip this step if you think you have Java. You will be prompted later if you don't already have it.
  4. Run npm start.
    The app consists of prestart and postinstall scripts that will ensure installation of npm dependencies and bower components. No need to install manually.


Hope this boostrap seed project works well.
